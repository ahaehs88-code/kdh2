# PULSE FITNESS (hc114.co.kr) 웹사이트 탐색 및 UI/UX 분석 리포트

본 리포트는 처음 방문한 사용자 관점에서 `http://hc114.co.kr` (PULSE FITNESS) 사이트를 전반적으로 탐색하며 수행한 UI 깨짐, 링크 오류(404 에러 등), 기능 작동 여부의 테스트 과정과 발견된 문제점을 정리한 결과물입니다.

---

## 1. 테스트 개요
- **테스트 대상**: [PULSE FITNESS (hc114.co.kr)](http://hc114.co.kr)
- **테스트 관점**: 처음 방문한 사용자 관점 (데스크톱 환경)
- **테스트 범위**: 메인 페이지 및 상단 메뉴(GNB)의 모든 링크와 인터랙션 기능
- **테스트 일시**: 2026년 6월 26일

---

## 2. 테스트 과정 및 상세 결과

메인 페이지를 시작으로 상단 헤더에 위치한 9개의 네비게이션 메뉴를 순차적으로 클릭하며 검증을 진행했습니다.

### ① 홈 (`#hero`)
- **결과**: 정상 작동
- **상세**: 클릭 시 메인 페이지 최상단(히어로 영역)으로 부드럽게 스크롤됩니다.

### ② 프로그램 (`#programs`)
- **결과**: 정상 작동
- **상세**: 클릭 시 프로그램 소개 섹션(`#programs`)으로 스크롤 이동합니다.

### ③ BMI 계산기 (`#bmi`)
- **결과**: 정상 작동
- **상세**: 클릭 시 BMI 계산기 섹션으로 스크롤 이동합니다. 키와 체중을 입력하고 분석 버튼을 눌렀을 때, 계산 결과가 정상적으로 화면에 표시되는지 확인했습니다. (예: 180cm, 80kg 입력 시 BMI 24.7로 정상 범위 판정 및 눈금 표시 확인)

### ④ 트레이너 (`#trainers`)
- **결과**: 정상 작동
- **상세**: 클릭 시 트레이너 소개 섹션으로 스크롤 이동합니다.

### ⑤ 멤버십 (`#pricing`)
- **결과**: 기능 미완성 (준비중)
- **상세**: 클릭 시 멤버십 섹션으로 스크롤 이동하나, 화면에는 **"멤버십 준비중입니다"**라는 문구가 노출되고 세부 요금제나 안내 사항은 표시되지 않습니다.

### ⑥ 전국 지점 찾기 (`gyms.html`)
- **결과**: 정상 작동 (신규 페이지 이동)
- **상세**: 클릭 시 지점 찾기 전용 페이지(`gyms.html`)로 정상 이동합니다. 텍스트 검색창에 "PULSE"를 입력하여 필터링하는 기능과 지역(서울, 부산 등) 드롭다운 필터링, 검색 초기화(Reset) 기능이 모두 오류 없이 올바르게 동작함을 확인했습니다.

### ⑦ 문의하기 (`#contact`)
- **결과**: 기능 미완성 (준비중)
- **상세**: 클릭 시 페이지 하단의 문의하기 섹션으로 이동하나, **"현재 온라인 예약 및 문의 서비스가 준비 중에 있습니다"**라는 문구가 노출되어 실제 예약이나 문의 접수는 불가능한 상태입니다.

### ⑧ 근육 정보 (`muscle.html`)
- **결과**: 정상 작동 (신규 페이지 이동)
- **상세**: 클릭 시 남자 근육 구조 가이드 페이지(`muscle.html`)로 정상 이동합니다. 근육 모형 이미지 위의 마커들을 클릭했을 때 해당 부위(예: 삼각근 등)의 설명 모달창이 정상적으로 팝업되며, 닫기 버튼도 정상 작동합니다.

### ⑨ 상담 예약 (`#contact`)
- **결과**: 정상 작동
- **상세**: 클릭 시 하단의 문의하기 섹션(`#contact`)으로 이동합니다.

## 3. 발견된 문제점 및 개선 제안

테스트 중 UI가 완전히 깨지거나 404 에러가 발생하는 등 치명적인 링크 오류는 발견되지 않았으나, 사용자 경험(UX) 향상을 위해 아래의 문제점들에 대한 개선이 필요합니다.

### 1. 메인 페이지 타이틀(Title) 불일치 오류
- **현상**: 메인 페이지(`index.html`)의 웹 브라우저 탭 타이틀이 브랜드명인 "PULSE FITNESS"가 아닌 **"공덕동"**으로 설정되어 있습니다. 
- **영향**: 사용자가 브라우저 탭을 보았을 때 사이트 정체성을 파악하기 어렵고 신뢰도가 저하될 수 있습니다.
- **해결 방안**: 다른 서브 페이지들과 일관되게 `<title>PULSE FITNESS - 홈</title>` 또는 적절한 대표 타이틀로 변경이 필요합니다.

### 2. 주요 핵심 서비스의 미오픈 (준비중) 상태
- **현상**: '멤버십'과 '문의하기/상담 예약' 기능이 준비 중인 상태로 표시되어 있습니다.
- **영향**: 사이트를 방문한 사용자가 요금을 확인하거나 상담 예약을 할 수 없어, 실제 고객 전환으로 이어지기 어렵습니다.
- **제안**: 빠른 시일 내에 간이 문의 폼이나 임시 연락처 정보라도 제공하는 것이 좋습니다.
